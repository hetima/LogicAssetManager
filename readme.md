#LogicAssetManager

LogicAssetManager is resource switcher for Logic Pro X.

##Features

- No needs backup original files
- Loads multiple theme together
- Variant options in single theme
- Customizing instrument icons

##System Requirements

- OS X 10.9
- Logic Pro X
